🎓 研究生招生

课题组主要围绕计算机视觉与多模态医学图像分析开展研究，重点关注口腔CBCT影像理解、三维视觉与深度学习方法等方向。欢迎对相关领域感兴趣、具备良好基础和科研意愿的同学报考。

课题组优势

  • 科研项目充足：主持多项国家级及省部级科研项目，研究方向稳定，课题来源明确。
  • 经费条件良好：具备充足科研经费，支持实验研究、论文投稿及学术交流。
  • 学术氛围活跃：持续开展科研工作并保持稳定论文产出，鼓励参与学术会议与合作研究。
  • 培养体系系统：从基础知识、代码能力到论文写作进行完整训练。
  • 指导投入充分：导师参与选题、实验与论文全过程，提供持续、细致的指导。

适合加入课题组的同学

  • 对科研有稳定兴趣，愿意持续投入时间与精力。
  • 具备一定数学基础与编程能力，能够主动学习新知识。
  • 具备耐心与执行力，能够反复实验、修改与完善工作。
  • 具备良好沟通能力和团队协作意识。

培养方式

  • 根据学生基础制定阶段性学习与研究计划。
  • 通过文献阅读、代码实现、项目实践与论文写作逐步提升能力。
  • 鼓励参与学术会议、汇报与合作研究。

基本要求

  • 具备良好的自我管理能力，能够保持稳定科研投入。
  • 具备基本编程能力，有 Python / PyTorch 经验者优先。
  • 遵守学术规范，认真对待实验与论文工作。
欢迎真正热爱科研、愿意踏实积累、希望系统提升科研能力的同学联系交流。

👨‍🔬 本科生科研

课题组欢迎有志于继续深造的本科生参与科研训练。本科阶段以能力培养为主，重点在于建立科研兴趣、掌握基础方法并提升编程与分析能力。

适合参与的同学

  • 对人工智能、计算机视觉或医学图像分析有明确兴趣。
  • 具备基础编程能力，愿意学习 Python 与深度学习框架。
  • 具备执行力，能够持续完成阅读、复现与实验任务。
  • 有继续读研或深造计划者更为适合。

主要收获

  • 了解科研基本流程（选题、文献、实验、总结）。
  • 提升编程能力与算法理解能力。
  • 优秀学生可参与论文撰写、项目研究及推荐深造。

基本要求

  • 能够保证稳定时间投入，而非短期参与。
  • 愿意从基础做起，认真完成各阶段任务。
  • 具备责任意识与良好沟通习惯。
本科科研更适合有明确目标、愿意长期投入并认真接受训练的同学。
